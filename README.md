# Customer Relationship Management

## Backend

mở cmd hoặc power shell tại thư mục root
'''
cd backend
mvn spring-boot:run
'''

### Frontend - reactJs

mở cmd hoặc power shell tại thư mục root
'''
npm install
cd frontend\crm
npm install
npm run dev
'''

Truy cập đến đường link http://localhost:5173/
